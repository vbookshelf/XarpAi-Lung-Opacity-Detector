FAQ

1- What does this app do?

This app automatically analyzes chest x-rays and draws bounding boxes around suspected opacities. Radiologists can then review the highlighted areas and use their clinical judgment to make a final diagnosis. This is a free diagnosis support tool.

2- How do I use it?

- Submit one or more chest x-rays in dicom, png or jpg format.
- Your images will be auto analyzed on submission.
- A list of image names will appear.
- The number of suspected opacities will also be displayed.
- Click an image name to see that image with the bounding boxes drawn in.
- Click anywhere on the image to hide or show the bounding boxes.

You'll find a few sample chest x-rays in the folder called sample_xray_images. You can use them to try the app.

4- How does this app work?

A fine tuned Faster R-CNN computer vision model analyzes each image. When opacities are detected, the model outputs their bounding box coordinates. These boxes are then drawn on the x-ray.
Please refer to the Model Card for more info.

5- What image formats are supported?

The app accepts dicom, png or jpg images. Tiff images are not supported.

6- Can this app analyze pediatric images?

Yes it can. But there wasn't a lot of pediatric images in the training data, therefore the performance on pediatric images may not be as good as adult images. The pediatric validation results are shown on the model card.

7- Does this app need an internet connection to work?

No. It's been designed to run offline. However, you will need an internet connection for the initial download and setup.

8- Will I need to have a computer with a GPU and tons of RAM?

A GPU would be faster, but it's not essential. This app runs fine on the CPU. You don't need lots of RAM either. An ordinary Mac or PC should be able to run this app without any issues.

9- Will the app always be free?

Yes it will always be free. This is an open source project.

There's no charge for using this app. But please submit feedback regarding bugs, things that need to be improved and the app's real world performance. You can leave feedback here.
